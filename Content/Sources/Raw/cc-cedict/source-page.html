CC-CEDICT download

The recommended way to get CC-CEDICT is to download the latest release version through the MDBG dictionary website:

  • MDBG Chinese-English dictionary

The latest non-verified version of CC-CEDICT can be downloaded for editing and reviewing purposes in the following formats:

  • ZIP format
  • GZip format

Beta: The latest non-verified version of CC-CEDICT can be downloaded for editing and reviewing purposes in the following formats:

  • Mixed version 1 and version 2 entries in ZIP format or GZip format
  • Version 1 entries upgraded to version 2 in ZIP format or GZip format (upgraded entries might not be fully valid version 2 entries)
